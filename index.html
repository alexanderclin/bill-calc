Add multiple purchasers for a single item with comma separated values. Use colon for specific splits.
Examples:
"Alice,Bob" : each gets 50% of the item price
"Alice:0.25,Bob:0.25,Charlie:0.50"

If "Total Pre Tax and Tip" is empty, it will use the calculated total price from items.

If "Total After Tax and Tip" is empty, it will use "Total Pre Tax and Tip".

(don't use this for anything important nothing is tested)
